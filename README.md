# 🚀 HR Analytics & Employee Attrition Dashboard

This project presents an interactive HR Analytics Dashboard built using **Microsoft Power BI** to analyze employee attrition and workforce trends.

---

## 🎯 Objective
To transform raw HR data into actionable insights and identify key factors influencing employee attrition.

---

## 📊 Key Features
- Interactive HR Dashboard with KPI cards  
- Employee-level analytics dashboard  
- Attrition analysis by:
  - Department  
  - Salary slab  
  - Job role  
  - Years at company  
  - Education  
- Dynamic slicers (Department, Gender, Age Group)  
- Cleaned and transformed data using Power Query  
- DAX measures for accurate calculations  

---

## 📌 Key Metrics
- Employees: 1470  
- Attrition: 237  
- Attrition Rate: 16.1%  
- Avg Salary: $6.5K  
- Avg Age: 36.92  
- Avg Years: 7.01  

---

## 🔍 Insights
- Highest attrition in Research & Development and Sales  
- Maximum attrition in salary < 5K  
- Early tenure employees are more likely to leave  
- Lower job levels show higher attrition  

---

## 🛠️ Tools Used
- Microsoft Power BI  
- Power Query  
- DAX
